# XYZRoguelike

## Описание

Это roguelike-проект на базе SFML с тремя локациями, боем с боссом и простым инвентарём. В текущей версии игра фокусируется на переходах между уровнями, уничтожении босса и использовании предметов.

## Что реализовано

1. **Несколько локаций**
   - Локации: `Ruins Entrance`, `Cave Hall`, `Crystal Vault`.
   - Переход на следующую локацию происходит после завершения текущей.

2. **Босс на каждой локации**
   - На каждой карте появляется отдельный босс.
   - Победа над боссом открывает следующую локацию.

3. **Инвентарь**
   - Хранение предметов реализовано через класс `Inventory`.
   - Использование предметов:
     - `H` — лечиться на `+25 HP`.
     - `B` — активировать временный `speed boost`.
   - Предметы выдаются при прохождении локаций.

4. **HUD и подсказки**
   - В HUD отображаются здоровье игрока, состояние босса, текущая локация и доступные предметы.
   - Подсказки показывают управление и текущую задачу.

## Управление

- `W`, `A`, `S`, `D` — движение.
- `Space` — атака.
- `H` — использовать зелье лечения.
- `B` — использовать зелье скорости.
- `N` — перейти на следующую локацию после победы над боссом.
- `R` — перезапустить уровень.

## Запуск

### Windows

1. Откройте `Game.sln` в Visual Studio.
2. Соберите решение в `Release`.
3. Запустите `XYZRoguelike`.

### Примечание по окружению

В текущем контейнере отсутствует Visual C++ toolchain, поэтому полноценная локальная сборка не подтверждена. Проверка идёт на уровне статического анализа через `get_errors` и инспекции кода.
