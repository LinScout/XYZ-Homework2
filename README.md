# XYZRoguelike

## Описание

Это учебный проект на базе SFML, в котором реализована логика прогулки по нескольким уровням, переходов между локациями и простого инвентаря. В текущей версии логика уровня уже построена на `levels.config`, а я усилил её следующими фичами:

- **Переходы между картами** через завершение текущего уровня.
- **Босс на каждой карте** — отдельный блок с повышенной стойкостью, который открывает доступ к следующей локации.
- **Инвентарь и использование предметов** — лечение и временный буст скорости мяча.
- **Состояние игрока** — здоровье, подсказки и текстовый HUD.

## Что добавлено

1. **Переход между локациями**
   - При полном уничтожении всех обычных блоков и босса игра переключает уровень или завершает игру победой.

2. **Босс на каждой карте**
   - Для каждого уровня создаётся отдельный `BossBlock`.
   - Победа над боссом считается завершением текущей локации и запускает переход на следующую.

3. **Инвентарь**
   - Хранение предметов через отдельный класс `Inventory`.
   - Использование предметов по клавишам:
     - `H` — лечиться.
     - `B` — активировать кратковременный boost скорости мяча.
   - Предметы можно получать как лут при разрушении блоков.

4. **Здоровье и повторный раунд**
   - Если мяч падает ниже платформы, игрок теряет одно здоровье.
   - При оставшемся запасе здоровья раунд перезапускается без выхода в Game Over.

## Запуск

### Windows

1. Откройте `Game.sln` в Visual Studio.
2. Соберите решение в `Release`.
3. Запустите `XYZRoguelike`.

### Примечание по окружению

В текущем контейнере отсутствует Visual C++ toolchain, поэтому локальная сборка через `dotnet msbuild` не проходит: MSBuild не может найти `Microsoft.Cpp.Default.props`. Сборка подтверждена только на уровне статического анализа через `get_errors`; исполняемый файл в этом окружении не создаётся.

## Архив с релизом

В текущем окружении полноценная Release-сборка недоступна из-за отсутствия MSVC. При необходимости я могу подготовить архив с исходниками и инструкцией для запуска на Windows, но для этого нужен доступ к полноценному инструментарию Visual Studio.
